#  DirBuster  -  Web Directory Scanner

> 
>  Author: **DUCVIET1622**

---

##  Giới thiệu

**DisBuster ** 

* Quét các đường dẫn (paths) ẩn trên website
* Phát hiện các endpoint tồn tại như `/admin`, `/login`, `/config`


---

##  Tính năng

*  Multi-thread (scan nhanh)
  
*  Hỗ trợ HTTP & HTTPS
*  Wordlist built-in (không cần file `.txt`)

##  Ví dụ

```text
Target: example.com
Paths: robots.txt,admin,login
```

Kết quả:

```text
[+] 200 /robots.txt
[+] 403 /admin
```

##  Công dụng

*  Tìm đường dẫn ẩn trên website
*  Phát hiện file nhạy cảm (.env, backup, config,...)
*  Học pentest / ethical hacking
*  Kiểm tra bảo mật website cá nhân
